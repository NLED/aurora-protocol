Install Instructions:

    Install npm packages if utilizing the server-node.js serial ports or TCPIP. 

    No install is required to use web serial in Chrome or Edge

    npm install


Usage Instructions:

    Open index.html in a web browser to start

    To use with a serial port:
        On Chrome and Edge Web Serial Ports can be used
        Or use NodeJS and run 'server-node.js'

    Use the example command user input elements to issue commands to connected NLED devices.
    View the source code to see how the command structure works. Cut/paste snippets or modify to suit.
